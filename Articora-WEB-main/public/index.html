¿Qué es Artícora?


Articora es el sitio donde puedes encontrar las mejores fuentes bibliograficas para tus investigaciones. Valoradas y calificadas para asegurarte una investigación de calidad.

Dentro de Artícora

Encontrar fuentes

Puedes encontrar fuentes bibliograficas publicadas por otros usuarios a los que les apasiona la investigación tanto como a ti.

Publicar tus fuentes

Ademas, tienes la posibilidad de publicar y guardar tus propias fuentes bibliograficas, así, para que los demas usuarios puedan expandir su conocimiento y conocer tus gustos.

Nuestra Misión.

Nuestra principal misión es formar una comunidad de investigadores que puedan apoyarse mutuamente al publicar fuentes bibliograficas de calidad y realmente utiles para cada investigador.
